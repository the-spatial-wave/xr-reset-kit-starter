# Models

Place your GLTF models here.

## Graffiti Chair

When the `chair-graffiti.glb` model is available:

1. Drop `chair-graffiti.glb` in this folder
2. Open `src/components/ChairPlaceholder.tsx`
3. Replace the export with the GraffitiChair component (instructions inside the file)
4. Run `npm run build` and verify 0 errors
