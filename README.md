Starter kit per costruire scene WebXR con React Three Fiber, Three.js e Vite.

---

## Stack

- React 18
- @react-three/fiber v8
- @react-three/drei
- @react-three/postprocessing v2
- Three.js 0.183+
- Vite 5 + TypeScript

---

## Avvio rapido

```bash
git clone https://github.com/the-spatial-wave/xr-reset-kit-starter
cd xr-reset-kit-starter
npm install
npm run dev
```

---

## Plugin Claude Code

Per creare nuove scene, aggiungere audio, video e luci con Claude:

```bash
claude plugin install github:the-spatial-wave/xr-spatial-kit-plugin
```
